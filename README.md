# House Price Predictor

## Setup

1. **Create virtual environment**

   - **Windows:**
     ```bash
     py -m venv venv
     ```
   - **Linux/MacOS:**
     ```bash
     python3 -m venv venv
     ```

2. **Install required packages**

   - **Windows:**
     ```bash
     pip install -r requirements.txt
     ```
   - **Linux/MacOS:**
     ```bash
     python3 -m pip install -r requirements.txt
     ```

3. **Create `.env` file and fill it as in `.env.example`**

4. **Run project**

   - **Windows:**
     ```bash
     py main.py
     ```
   - **Linux/MacOS:**
     ```bash
     python3 main.py
     ```
